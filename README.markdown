## 我的个人日志系统

## 步骤

1. 安装git
2. rvm , ruby (1.9.3) 和 (bundle)
3. clone 代码
'''bash
$ git clone https://github.com/imathis/octopress.git myblog
'''
4. 安装依赖的ruby库
'''bash
cd myblog
bundle install 
'''
5. 部署blog的theme等
'''bash
rake install
'''
6. 启动
'''bash
rake preview
'''
